Built with HTML, CSS, and JavaScript, my portfolio website is designed to be responsive, working across a variety of devices.

The Challenge

Primarily built with a grid layout it also features Javascript-controlled transitions between pages and navigational controls to make it easier for the user to view my projects.

The Design

I opted for a minimalistic design to ensure that the focus stays on the content. The site's made with pure CSS primarily with the grid layout to ensure that the design stays consistent on all screen sizes.

In addition, my portfolio website also features a projects page with smooth scroll and navigation.
