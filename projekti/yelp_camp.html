Production app for campground creation, rating and viewing

Yelp Camp

2022

HTML // Bootstrap 5 // JavaScript // MongoDB // Node.js

Yelp Camp is a full-stack web application which main functions are campground viewing and creating. The user has the option to leave a rating or edit and delete their own campgrounds.

The Challenge

In this personal project, my primary focus was to create a full-stack web application that provides a secure and responsive environment for the user.

The Design

Front-end of the website was built with the CSS framework Bootstrap 5 because of its ease of use and responsive features.

The main focus of the design was to display all the necessary information to the user.

Authentication and authorization are encrypted and stored via a Node.js tool called Passport.

Node.js was used for server-side scripting.Express and MongoDB were employed to create a full-stack JavaScript application. As for the deployed sites database I opted to use Mongo Atlas.

I implemented the authentication with measures in place to ensure common security issues, such as SQL Injection and XSS.

After logging in the user also has an option to upload an image to the website, which will be stored in a cloud database tool called Cloudinary.

Lastly, I utilized the Mapbox API to both determine the location of campgrounds and pin them on a map. Furthermore, an interactive cluster map of all the campgrounds was also created using the API.
