AI Image Generation App

2023

React // Tailwind CSS // Javascript // MongoDB // Node.js

In this website the user has the option to create, view, search and download artificial inteligence generated images that are based on unique user given prompts.

The Challenge

In this project the most important thing was to ensure a working connection with the OpenAI API. Once that was done, I needed a place to store user creations and because of my prior knowledge of MongoDB I opted to use it for it's reliable storage capabilities.

The Design

I used Tailwind CSS for the sites front-end because of its ease of use and responsive features. The website is fully responsive on all screen sizes.

In the "Create" page the user has to enter a name, prompt or use one of the many pre-determined prompts with the "Surprise me" button and generate an AI image. After they're satisfied with the end result they can share it on the site for others to see.

The site is built with React.js, Node.js, Express.js and MongoDB for storing information. The site is based on the use of OpenAI's DALL-E API which is a A deep learning model that generates images from text input. I also used a cloud-based image storage service Cloudinary to store user generated images.
